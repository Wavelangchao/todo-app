# Changelog

## [2.0.0] - 2025-06-29

### Changed
- **UI 全面重构** — Linear × Notion 极简商务风格
- **全宽自适应布局** — 侧边栏 + 命令栏 + 任务列表
- **侧边栏** — 分类导航、可折叠、计数徽章
- **命令栏** — Cmd+K 风格搜索 + 快捷操作
- **任务行** — 去卡片化，1px 分隔线，hover 显示操作
- **优先级** — 左侧 3px 色条替代圆点
- **配色** — 全新浅色/暗黑配色方案
- **字体** — Inter + Noto Sans SC 字体组合

### Added
- 分类侧边栏（可折叠，hover 展开）
- 命令栏快捷操作（添加/清除/导出）
- 自定义分类支持
- 键盘快捷键：Ctrl+K 命令栏, Ctrl+N 新建, Ctrl+B 折叠

### Fixed
- 暗黑模式下 date input 配色

## [1.0.0] - 2025-06-29

### Added
- 初始化待办事项应用
- 添加/删除/编辑任务
- 优先级、分类、截止日期
- 搜索、筛选、排序
- 深色模式切换
- JSON 导入/导出
- 撤销删除
