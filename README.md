# Todo App

> **v2.0** — Linear × Notion 极简商务风格全宽自适应布局

一个纯前端待办事项应用，无需任何依赖。

## ✨ 功能

- 📋 **侧边栏导航** — 分类管理、任务计数、可折叠
- ⌨️ **命令栏** — Ctrl+K 搜索 + 快捷操作  
- ✅ **任务管理** — 添加/编辑/删除、优先级、截止日期
- 🎨 **深色模式** — 浅色/暗黑一键切换
- 🔍 **筛选排序** — 按状态/优先级/日期/名称
- 📤 **导入导出** — JSON 格式备份
- ↩️ **撤销删除** — 误删可恢复
- 📱 **响应式** — 桌面 + 移动端适配

## 🚀 使用

直接用浏览器打开 `index.html` 即可。

### 快捷键

| 快捷键 | 操作 |
|--------|------|
| `Ctrl + K` | 打开命令栏 |
| `Ctrl + N` | 新建任务 |
| `Ctrl + B` | 折叠/展开侧边栏 |

## 🛠 技术栈

HTML5 + CSS3 (CSS Variables) + Vanilla JavaScript + localStorage
