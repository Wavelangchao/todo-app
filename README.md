# ✅ 待办事项 (Todo App)

一个功能完善的单页待办事项应用，纯 HTML/CSS/JS，无需任何依赖，双击即用。

## ✨ 功能

- **优先级** — 高🔴 / 中🟡 / 低🟢 三档
- **截止日期** — 逾期自动标红提醒
- **分类标签** — 个人 / 工作 / 学习 / 生活
- **搜索** — 支持 `Ctrl+K` 快速聚焦
- **深色模式** — 一键切换，自动记忆
- **进度条** — 可视化完成百分比
- **内联编辑** — 双击任务文字直接修改
- **撤销删除** — 删除后可一键恢复
- **导出/导入** — JSON 格式备份，支持合并
- **多维度排序** — 时间 / 优先级 / 截止日期 / 名称
- **本地存储** — 所有数据保存在浏览器 localStorage

## 🚀 使用

1. 双击 `index.html` 在浏览器中打开
2. 开始添加任务！

或部署到任意静态服务器：

```bash
npx serve .
# 或
python -m http.server 8080
```

## ⌨ 快捷键

| 按键 | 功能 |
|------|------|
| `Enter` | 添加任务 / 确认编辑 |
| `Ctrl+K` | 聚焦搜索框 |
| `Esc` | 清空搜索 |
| 双击任务 | 编辑文字 |

## 📦 技术栈

纯前端，零依赖：HTML + CSS + JavaScript (localStorage)
